# Курсы драгоценных металлов

 ## Описание
Получение курса драгоценных металлов - золото, серебро, палладий, платина - и их отображение.

## Технологии
 - **C#**
 - **.NET Framework**
 - ** API Веб-сервиса ЦБ РФ - http://www.cbr.ru/DailyInfoWebServ/DailyInfo.asmx**
